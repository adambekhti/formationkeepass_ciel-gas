Introduction

Pourquoi KeePass va vous sauver ?

Fini les post-it sur l'écran ou le mot de passe unique pour tout. KeePass est votre coffre-fort ultra-blindé personnel.

Une seule chose à retenir : Votre mot de passe maître donne accès à tout le reste.
Chiffrement Total : Même si on vole votre fichier, il est illisible sans la clé.
Installation

Installation Professionnelle

Pour CIEL-GAS, nous utilisons la version KeePass 2.x. Suivez ces étapes :

Téléchargement : Allez sur keepass.info et prenez la version Portable (évite les traces système).
Vérification : Toujours vérifier le hash SHA-256 du fichier pour éviter les malwares.
Conseil : Allez dans Fichier > Paramètres > Sécurité et choisissez Argon2id. C'est le réglage le plus solide contre les GPU.
Pratique

La sécurité de votre Clé

Testez la force de votre futur mot de passe maître ci-dessous :

  • 12+ caractères
  • Majuscule
  • Chiffre
  • Caractère spécial
Utilisation

Apprendre à utiliser KeePass

1. Créer une entrée

Cliquez sur l'icône de clé dorée. Remplissez le nom du site et l'identifiant. KeePass génère un mot de passe complexe pour vous.

2. Le raccourci magique

Ne tapez plus vos mots de passe ! Utilisez CTRL + V (Auto-Type) sur la page de connexion. KeePass remplit tout seul.

Sécurité : KeePass efface votre presse-papier automatiquement après 12 secondes.
Résilience

A. Taper très vite au clavier.
B. Utiliser le raccourci Auto-Type (CTRL+V).
