Module 01

Architecture & Déploiement

Fondamentaux Tech

KeePass est un coffre-fort Zero-Knowledge. CIEL-GAS conserve l'exclusivité des clés de déchiffrement.

  • Isolation : Aucune base de données sur le cloud public.
  • Format KDBX 4.0 : Support des pièces jointes et métadonnées chiffrées.

Standard Interne

Le déploiement doit suivre le standard établi pour la flotte de techniciens :

  • Binaire portable (éviter l'installation MSI).
  • Plugin de langue FR obligatoire.
Module 02

Sécurité & Cryptographie

Chiffrement AES-256

Standard de l'industrie pour la protection des données gouvernementales.

// Processus AES-256
[Input] ClearText_Pass
[Key] Master_Key (256-bit)
[Rounds] 14 Cycles de transformation
[Output] CipherText_Encrypted_Data

Protection Argon2

Configuration recommandée pour CIEL-GAS afin de contrer les attaques par force brute (GPU).

  • KDF : Utiliser Argon2id.
  • Délai : Paramétrer sur 1 seconde de calcul.

Module 04

Résilience & Sauvegardes

La règle d'or 3-2-1

Pour assurer la continuité de service de CIEL-GAS :

  • 3 copies des bases de données.
  • 2 supports différents (Local / NAS).
  • 1 copie hors-site (Coffre physique ou Cloud chiffré).
